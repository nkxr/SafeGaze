# SafeGaze
SafeGaze คือระบบ Driver Monitoring System (DMS) อัจฉริยะที่พัฒนาด้วย Python และ Computer Vision เพื่อช่วยป้องกันอุบัติเหตุจากการหลับในและการละสายตาขณะขับขี่ ระบบทำงานแบบ Real-time โดยใช้ AI (MediaPipe) วิเคราะห์ใบหน้าเพื่อตรวจจับอาการง่วงนอน (Drowsiness), การหาว (Yawning), และการเหม่อลอย (Distraction)
